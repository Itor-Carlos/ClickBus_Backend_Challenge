# ClickBus_Backend_Challenge
Simple challenge to test my skills on building APIs

## Technologies
 
- Java
- Spring Boot
- MySQL
- JPA
- OpenSwagger
- H2
- JUnit

## Features

- Save a Place
- Get all Places
- Get a specific Place
- Uptade a specific Place
- Delete a specific Place
- Search Places

## Requirements

1. Java - 1.8.x

2. Maven - 3.x.x

3. MySQL - 5.x.x

  + CORS configuration
  + Profiles (test,dev)
  + Unit Tests and Integration Tests
  + Docker
